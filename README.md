# 智慧旅遊規劃 App（可執行成品）

這個專案已提供可直接開啟使用的前端成品，支援：

- 輸入旅遊地點、日期與時間區間
- 自動規劃建議路線（景點順序）
- 顯示交通工具、每段時間與花費估算
- 整合當地當日天氣（Open-Meteo）
- 依偏好（省錢/省時/避雨/少走路）重新計算結果

## 快速開始

```bash
python3 -m http.server 4173
```

開啟 `http://localhost:4173`。

## 檔案說明

- `index.html`：頁面與表單
- `styles.css`：介面樣式
- `app.js`：路線規劃、成本估算、天氣整合

## 實作說明

- 天氣與城市座標使用 Open-Meteo API（免金鑰）
- 內建 Tokyo/Taipei/Paris 景點資料（可再擴充）
- 路線採「偏好加權排序 + 時間窗裁切」快速產生可執行行程
